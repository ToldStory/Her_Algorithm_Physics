# Concrete Statements After the First U(1) Test

This document separates theorem-level claims, U(1)-specific computational claims, and claims that remain conjectural.

## Theorem-level statement

Let `G` be a group. Define relational links

\[
U_{AB}=A^{-1}B.
\]

Then for all `A,B,C in G`,

\[
U_{AB}^{-1}U_{AC}=U_{BC}.
\]

This is not a physics claim. It is an algebraic identity.

## Gauge-language translation

On a graph with site frames `T_i`, define complete pure-gauge links

\[
U_{ij}=T_i^{-1}T_j.
\]

Then the Her re-anchoring identity becomes

\[
U_{ij}^{-1}U_{ik}=U_{jk}.
\]

The triangle residual is

\[
R_{ijk}=U_{ij}^{-1}U_{ik}U_{jk}^{-1}.
\]

For a pure gauge field, `R_ijk = e`.

## U(1)-specific result

For `U(1)` links generated by site phases

\[
U_{ij}=e^{i(\alpha_j-\alpha_i)},
\]

all tested triangle residuals vanish to floating-point precision.

Under local gauge transformations

\[
U_{ij}\mapsto h_i^{-1}U_{ij}h_j,
\]

`U(1)` residuals are invariant, not just covariant, because `U(1)` is Abelian.

## Perturbation result

Sparse phase perturbations of links create nonzero triangle residuals. Ranking links by residual participation can recover the corrupted links in simple synthetic cases.

This is only an algorithmic diagnostic result. It is not evidence of a new physical law.

## Conjectures still open

1. In non-Abelian groups, residuals should transform by endpoint conjugation rather than scalar invariance.
2. Triangle residuals should be compared against plaquette holonomy before making any curvature statement.
3. Residual participation may localize corrupted links, but recall must be measured across many seeds, lattice sizes, noise levels, and clustered fault patterns.
4. Non-invertible settings remain outside the exact theorem unless a new generalized structure is defined.

## Strongest safe public wording

> Her Algorithm is an exact relational re-anchoring identity in groups and groupoids. In finite gauge-like link fields, the same identity gives a triangle consistency residual. For U(1) pure gauges this residual vanishes to numerical precision and is invariant under local U(1) gauge transformations. Perturbed synthetic links create nonzero residuals that can be used as preliminary localization signals.
