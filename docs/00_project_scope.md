# Project Scope

Core statement:

\[
\boxed{\text{Her Algorithm maps how relations change when the origin changes.}}
\]

Exact domain:

- groups,
- groupoids,
- invertible matrices,
- nonzero complex ratios,
- phase groups such as \(U(1)\),
- chart atlases with invertible transition functions.

Caution:

> This is a precise relational structure with promising gauge-theoretic tests, not yet a complete physical theory.
